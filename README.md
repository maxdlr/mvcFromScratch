# MVC from scratch

## Démarrage du projet

### Installer les packages

```shell
composer install
```

### lancer le serveur avec la commande

```shell
composer ss
```

### Créer un fichier ``.env.local`` à la racine du projet

Avec les paramètre suivant dedans:

```dotenv
DB_USER="username"
DB_PASSWORD="password"
```
